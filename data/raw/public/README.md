# Public Raw Data Note

This anonymous review repository does not redistribute the full original MAVEN-ERE or ChronoQA raw archives.

To regenerate public-benchmark intermediates from scratch, please obtain the original public datasets from their official release sources and place them in this directory using the filenames expected by the configs, e.g. `MAVEN_ERE.zip` and `ChronoQA_raw_data.zip`.

The repository already includes the fixed paper-aligned evaluation subset manifests used in the final experiments.
