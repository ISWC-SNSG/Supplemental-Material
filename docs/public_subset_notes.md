# Public Subset Notes

## MAVEN-ERE Causal-Temporal 90-Document Evaluation Subset
This subset is used only for component-level validation of query-driven local support-structure construction and risk-oriented causal-temporal path selection. It is not used for full-pipeline answer/rationale/uncertainty evaluation. The released files include the fixed subset manifest and paper-aligned reported-summary artifacts.

## ChronoQA Temporal-Balanced 90-Example Final Evaluation Subset
This subset is used only as a fixed diagnostic subset for component-level validation of the temporal-sensitive component of path selection. It is not used as a full-pipeline benchmark. The final reported ChronoQA results follow the revised metric definitions documented in `docs/chronoqa_metric_protocol.md` and can be recomputed from `artifacts/paper_results/chronoqa_90_final_per_task.csv`.

## Redistribution note
This repository releases the fixed subset manifests and evaluation metadata, not full mirrors of the original public raw datasets.
